    # element UI 十个小细节

    建议大致浏览一遍官方文档 (opens new window)后再看此篇文章，主要介绍一些关于 element UI 一些值得关注的细节(2.10.1)

    # Layout 布局 (opens new window)

    参数 说明
    基于断点的隐藏类 适用于多端适配

    # Input 输入框 (opens new window)

    参数 说明
    focus 使 input 获取焦点

    TIP

    这里需要注意的两点：

    • 如果你需要使用这个方法，一定要在组件加载完毕后调用，如果在 created 钩子里面调用时不起作用的，具体请看Vue.nextTickt
    • 如果需要使用 @keyup 监听键盘事件的话，一定要加上 .native 修饰符,具体原因可以参考官方文档 (opens new window)

    # Pagination 分页 (opens new window)

    参数 说明
    hide-on-single-page 只有一页时是否隐藏

    # Message 消息提示 (opens new window)

    参数 说明
    offset Message距离窗口顶部的偏移量

    # MessageBox 弹框 (opens new window)

    参数 说明
    lockScroll 是否在 MessageBox 出现时将 body 滚动锁定
    closeOnClickModal 是否可通过点击遮罩关闭 MessageBox

    # Select 选择器 (opens new window)

    参数 说明
    default-first-option 在输入框按下回车，选择第一个匹配项。需配合 filterable 或 remote 使用（可以提升不错的用户体验）

    # Cascader 级联选择器 (opens new window)

    刚开始两年的 element UI 是不支持级联多选的，但是最近更新支持了

    # Table 表格 (opens new window)

    刚开始两年的 element UI 是不支持树状表格的，但是最近更新支持了

    # NavMenu 导航菜单 (opens new window)

    参数 说明
    unique-opened 是否只保持一个子菜单的展开
