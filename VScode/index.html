# 使用技巧

  • Ctrl + D

    快速选择多个单词，并进行多行编辑，多用于重命名变量，当然你也可用 F2

  • Shift + Alt + 鼠标点击拖动选择多行

    快速选择多行，进行多行编辑，在特殊场景下会很好用

  • Ctrl + Shift + D

    选择单行多行直接复制在下方

  • 电脑配置过低时候会觉得卡，可以设置 settings.json "search.followSymlinks": false,

    设置路径 文件 -> 首选项 -> 设置

  • Ctrl + P

    快速查找文件，这个在 chrome 浏览器 F12 里面 Sources同样适用

  • Ctrl + Shift + P

    各种扩展命令快捷搜索

  • Ctrl + ~

    打开终端

  • Ctrl + / Ctrl + A Ctrl + S

    注释代码 全选 保存
